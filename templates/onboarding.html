Install Root CA Certificate

All {{ app_title }} tools use HTTPS with a custom Root CA. Without this certificate your browser will show a security warning. After installation your device will automatically trust all internal domains.

Download Root CA
  1. 1. Double-click the downloaded root-ca.crt file
  2. 2. Keychain Access opens — choose the System keychain (not Login)
  3. 3. Double-click the certificate → open Trust
  4. 4. Set When using this certificate: to Always Trust
  5. 5. Close the window — enter your macOS password if prompted

Restart Safari/Chrome after installation.

  1. 1. Double-click root-ca.crt
  2. 2. Click Install Certificate
  3. 3. Choose Local Machine → Next
  4. 4. Select Place all certificates in the following store
  5. 5. Click Browse → choose Trusted Root Certification Authorities
  6. 6. Click Finish and confirm the security warning

Restart Edge/Chrome after installation.

Debian/Ubuntu/Mint:

sudo cp root-ca.crt /usr/local/share/ca-certificates/
sudo update-ca-certificates

Fedora/RHEL/Arch:

sudo cp root-ca.crt /etc/pki/ca-trust/source/anchors/
sudo update-ca-trust

Firefox uses its own store — also import via Settings → Privacy → View Certificates.
